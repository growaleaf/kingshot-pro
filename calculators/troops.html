How to read event point rates

Each training event (HoG, KvK, TSG) awards points per troop trained at a rate that scales with troop tier. T4 troops typically earn more points per troop than T3, but T3 trains faster — the pts/hour comparison tells you which is more efficient for a given event target.

To find your current event rates: open the event in-game → tap the reward info tab → look for "Training points per troop" or similar. Enter those values above for exact results.
